MY-136 Monitor hladiny / tlaku

Desktopová aplikácia na monitorovanie snímača tlaku / hladiny MY-136 cez Modbus TCP (RS485 snímač → prevodník RS485-to-Ethernet). Vizualizácia nádrže, MIN/MAX alarmy, história, e-mailové notifikácie.

Python 3 · pymodbus >= 3.6.0 · tkinter · pystray

Funkcie
Čo aplikácia robí
  • Zapamätá si poslednú IP a port (uložené vedľa aplikácie)
  • Stabilné čítanie cez jedno I/O vlákno; automatické zotavenie po výpadku — čistý reconnect, indikátor sa sám vráti na zelenú
  • Vizualizácia nádrže s MIN/MAX alarmami a prepínačom SK/EN
  • História hladiny — graf s voliteľným rozsahom, priebežný záznam a export do CSV; hover zobrazí dátum, čas a hodnotu
  • E-mailové notifikácie — pri výpadku komunikácie > 1 min a pri prekročení MIN/MAX rozsahu (vrátane obnovenia)
  • Beh na pozadí cez systémovú tray ikonu (Zobraziť / Ukončiť)
Rýchly štart
  1. Stiahnite a spustite aplikáciu (.exe alebo python level_sensor_monitor.py)
  2. Zadajte pripojenie: IP 192.168.1.1, Port 502, Slave ID 1
  3. Kliknite na Pripojiť — nameraná hodnota a nádrž sa začnú obnovovať
  4. Bez senzora? Spustite python simulator.py --port 502 a pripojte sa na 127.0.0.1

Simulátor (bez fyzického snímača)

Testovací Modbus TCP server emulujúci snímač MY-136.

Príkaz Popis
python simulator.py --port 502 štandardný Modbus port
python simulator.py --mode sine plynulá sínus vlna 0↔2000 mm
python simulator.py --speed 2.0 rýchlejšia animácia
Modbus registre (FC03 čítanie / FC06 zápis)
Adresa Popis FC Prístup
0 Slave ID FC03 len čítanie
1 Baud rate (3 = 9600 bps) FC03 len čítanie
2 Jednotka tlaku — 0=MPa … 8=mmH₂O FC03 FC06 čítanie / zápis
3 Desatinné miesto — 0=/1 · 1=/10 · 2=/100 · 3=/1000 FC03 FC06 čítanie / zápis
4 Nameraná hodnota (raw) — deliť deliteľom reg. 3 FC03 len čítanie
5 Nulový posun (zero offset) FC03 FC06 čítanie / zápis
